# DepartmentsEmployeesAPI

A simple .NET 8 Web API project for managing Departments and Employees.  
It includes CRUD operations, Entity Framework Core integration, and a clean layered structure.

## Features
- Manage departments and employees with full CRUD APIs  
- Follows clean architecture (Controller → Service → Repository → DbContext)  
- Uses Entity Framework Core for data access  
- AutoMapper for mapping between entities and DTOs  
- Validation using data annotations  
- Asynchronous methods for better performance  

## Technologies Used
- .NET 8 Web API  
- C#  
- Entity Framework Core  
- SQL Server  
- AutoMapper  
- Swagger UI
